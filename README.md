# PI - 2º Semestre

## Sobre o projeto
Este projeto integrador (PI) do 2º semestre tem como objetivo o desenvolvimento de um **website de fotografias** para nossa cliente **Dany Cardozo**.

A proposta do sistema é criar uma plataforma visual, organizada e funcional para apresentar fotografias, divulgar o trabalho da cliente e facilitar o contato com possíveis clientes interessados em seus serviços.

## Objetivo do projeto
Desenvolver um website moderno, responsivo e intuitivo, capaz de:

- Exibir o portfólio fotográfico da cliente
- Apresentar informações sobre os serviços oferecidos
- Facilitar o contato com clientes
- Organizar o conteúdo de forma visual e atrativa
- Garantir uma boa experiência de navegação em diferentes dispositivos

## Tecnologias utilizadas
O projeto será desenvolvido utilizando as seguintes tecnologias:

- **HTML** - estrutura das páginas
- **CSS** - estilização do site
- **Bootstrap** - responsividade e componentes visuais
- **JavaScript** - interatividade no front-end
- **PHP** - lógica de back-end
- **MySQL** - armazenamento e gerenciamento de dados

## Estrutura do projeto
O sistema será dividido em partes para melhor organização e desenvolvimento:

- **Front-end:** interface visual do site
- **Back-end:** lógica de funcionamento e processamento
- **Banco de dados:** armazenamento das informações
- **Documentação:** descrição técnica e acadêmica do projeto
- **Casos de uso:** levantamento das funcionalidades do sistema

## Possíveis funcionalidades
Entre as funcionalidades previstas para o website, estão:

- Página inicial com apresentação da fotógrafa
- Galeria de fotos
- Página sobre a profissional
- Página de serviços
- Formulário de contato
- Integração com redes sociais
- Área administrativa futura para gerenciamento de conteúdo

## Público-alvo
O website será voltado para:

- Pessoas interessadas em contratar serviços fotográficos
- Clientes que desejam conhecer o portfólio da fotógrafa
- Usuários que buscam contato rápido e fácil com a profissional

## Resultado esperado
Ao final do projeto, espera-se entregar um website funcional, visualmente agradável e de fácil utilização, representando profissionalmente o trabalho da cliente e atendendo às necessidades levantadas no PI.
